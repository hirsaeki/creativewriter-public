# Public Release Notes

Below are the change notes synced to the public release. For the full release, see the GitHub release page.

## Change Notes

### Features
- New Inspector module with Cliché Analyzer (beta) for text quality insights.
  - Components and services under `src/app/inspector/**` (routing, cache, analysis).
- New shared Model Selector component for choosing AI models.
  - `src/app/shared/components/model-selector/*`

### Refactor
- UI components migrated to `src/app/ui` and imports updated.
- Removed deprecated `core.module.ts` in favor of standalone Angular setup.
  - Updated `app.config.ts` and `app.routes.ts` accordingly.

### Improvements
- Editor: refinements in ProseMirror editor service and Story Editor behavior.
- Database: stability tweaks in PouchDB sync (`database.service.ts`).
- Services: minor updates to header navigation and custom background syncing.
- Types and markup: `pouchdb.d.ts` and `index.html` adjustments.

### Docs
- Added `RELEASE_NOTES_v1.4.202509021127.md` and `docs/improvement-report-2025-09-03.md`.

### Build
- Production build verified. Known CommonJS warnings from third‑party libs (e.g., canvg, pouchdb) remain but don’t affect output.
